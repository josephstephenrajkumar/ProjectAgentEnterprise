# ProjectAgentEnterprise Option 1 Refactor Package

This package is a complete Antigravity handoff to refactor the current OpenClaw-based repository into:

- React frontend
- FastAPI backend
- LangGraph agent orchestration
- LlamaIndex RAG ingestion/retrieval
- SQL/RAG router agents
- SQL learning layer
- Versioned project financial forecast database
- Human-in-the-loop autonomous agents

Start with:
`manifests/antigravity_refactor_manifest.yaml`

- `ARCHITECTURE.md`
- `migrations/001_agentic_financial_forecast_schema.sql`
- `manifests/agent_registry.yaml`
- `docs/ROUTER_AGENT_DESIGN.md`
- `docs/SQL_LEARNING_LAYER.md`
- `backend/app/services/migration_loader.py`
